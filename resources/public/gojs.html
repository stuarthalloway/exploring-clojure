Go blocks provide threadlike concurrency for JavaScript

          
(go (while true
  (<! (sleep 250)) 
  (>! c "Red")))
Go 
          
(go (while true
  (<! (sleep 1000)) 
  (>! c "Blue")))
Go 
          
(go (while true
  (<! (sleep 2000))
  (>! c "Green")))
